response\-With\-Url\-:with\-Params\-:error\-:

Performs a synchronous load of the given request, returning the formatted json object.

This method initiates a synchronous request, identifies the error status, and based on error status, it will parse the json object received into appropriate Objective C object and returns it.


\begin{DoxyParams}{Parameters}
{\em api} & The api to request. It wont expect the complete U\-R\-L. Only the last part of the U\-R\-L is expected here. Inside the method it will prepare the full U\-R\-L based on the W\-E\-B\-A\-P\-I\-\_\-\-U\-R\-L defined in U\-R\-L\-Constants\\
\hline
{\em params} & The input parameters to be sent to P\-O\-S\-T request\\
\hline
{\em error} & Out parameter (may be N\-U\-L\-L) used if an error occurs while processing the request. Will not be modified if the load succeeds.\\
