responseWithUrl:withParams:error:

Performs a synchronous load of the given request, returning the formatted json object.

This method initiates a synchronous request, identifies the error status, and based on error status, it will parse the json object received into appropriate Objective C object and returns it.

Parameters
apiThe api to request. It wont expect the complete URL. Only the last part of the URL is expected here. Inside the method it will prepare the full URL based on the WEBAPI_URL defined in URLConstants
paramsThe input parameters to be sent to POST request
errorOut parameter (may be NULL) used if an error occurs while processing the request. Will not be modified if the load succeeds.
Returns
The formatted json object.
